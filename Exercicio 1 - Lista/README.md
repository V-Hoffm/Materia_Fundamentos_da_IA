📁 Estrutura das Atividades
✅ Atividade 1 — Biblioteca math
Uso das principais funções matemáticas do módulo math:

Cálculo de cosseno e seno com m.cos() e m.sin()
Raiz quadrada com m.sqrt()
Logaritmo na base 2 com m.log2()


✅ Atividade 2 — Biblioteca calendar
Uso do módulo calendar para operações com datas:

Verificação de ano bissexto com isleap()
Descoberta do dia da semana de uma data com weekday() + day_name[]
Identificação do primeiro dia do mês com monthrange()


✅ Atividade 3 — Variáveis e Tipos
Exploração de tipos de dados em Python:

Diferença entre int e float
Operações aritméticas e conversão implícita de tipos
Cálculo de IMC usando função e math.pow()


✅ Atividade 4 — Operadores
Uso de operadores lógicos, relacionais e aritméticos:

Operadores >, <, ==, !=, >=
Operadores lógicos and, or
Classificação do IMC com estrutura if / elif / else


✅ Atividade 5 — Strings
Manipulação de strings em Python:

Fatiamento com índices [inicio:fim]
Inversão de string com [::-1]


✅ Atividade 6 — Tipos Complexos (Listas e Dicionários)
Trabalho com estruturas de dados:

Lista de listas com dados de PIB mundial
Adição de elementos com .append()
Dicionário com .keys(), .values() e sum()
Adição programática da chave total ao dicionário


🛠️ Tecnologias Utilizadas

Python 3.14.3
Módulo math
Módulo calendar
